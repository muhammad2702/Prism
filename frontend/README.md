# SABCEMM Frontend Dashboard

Professional presentation layer for the SABCEMM (Agent-Based Computational Economic Model) simulation framework.

## Features

### 📊 Advanced Visualizations
- **Price Evolution Charts** - Track price movements over time with interactive area charts
- **Returns Analysis** - Visualize returns distribution and time series
- **Rolling Volatility** - Monitor volatility changes with rolling window calculations
- **Statistical Distributions** - Histogram visualization of return distributions

### 📈 Quantitative Finance Metrics
- **Sharpe Ratio** - Risk-adjusted return calculation
- **Maximum Drawdown** - Identify worst peak-to-trough declines
- **Skewness & Kurtosis** - Measure distribution asymmetry and tail risk
- **Volatility Analysis** - Standard deviation and variance metrics
- Total Return, Min/Max Price tracking

### ⚙️ Simulation Control
- **XML Parameter Editor** - Edit simulation configurations in real-time
- **Template Management** - Load and modify existing simulation templates
- **Run Simulations** - Execute new simulations from the frontend
- **Live Updates** - Monitor simulation progress and results

### 📁 Data Management
- **Simulation Browser** - View all historical simulation runs
- **CSV Data Export** - Access all collected data points
- **Metadata Tracking** - Detailed simulation metadata and timestamps

### 🎨 Professional UI/UX
- Dark mode optimized for extended viewing
- Glassmorphism and gradient effects
- Smooth animations and transitions
- Responsive design for all screen sizes
- Premium quant finance aesthetic

## Getting Started

### Installation

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Start the backend API** (in separate terminal):
   ```bash
   cd ../server
   node api.js
   ```

4. **Access the application:**
   Open [http://localhost:5173](http://localhost:5173) in your browser
